# GSD 문서

Get Shit Done(GSD) 프레임워크를 위한 문서 모음입니다. AI 코딩 에이전트를 위한 메타 프롬프팅, 컨텍스트 엔지니어링, 스펙 기반 개발 시스템을 한국어 우선으로 안내합니다.

## 문서 인덱스

| 문서 | 대상 | 설명 |
|----------|----------|-------------|
| [Architecture](ARCHITECTURE.md) | 기여자, 고급 사용자 | 시스템 아키텍처, 에이전트 모델, 데이터 흐름, 내부 설계 |
| [Feature Reference](FEATURES.md) | 모든 사용자 | 요구사항까지 포함한 전체 기능 문서 |
| [Command Reference](COMMANDS.md) | 모든 사용자 | 명령어 문법, flags, 옵션, 예제를 한곳에 정리한 문서 |
| [Configuration Reference](CONFIGURATION.md) | 모든 사용자 | 전체 config schema, workflow 토글, 모델 프로필, git branching |
| [CLI Tools Reference](CLI-TOOLS.md) | 기여자, 에이전트 작성자 | 워크플로와 에이전트가 사용하는 `gsd-tools.cjs` API |
| [Agent Reference](AGENTS.md) | 기여자, 고급 사용자 | 15개 전문 에이전트의 역할, 도구, spawn 패턴 |
| [User Guide](USER-GUIDE.md) | 모든 사용자 | 워크플로 사용법, 문제 해결, 복구 가이드 |
| [Context Monitor](context-monitor.md) | 모든 사용자 | 컨텍스트 창 모니터링 hook 구조 |
